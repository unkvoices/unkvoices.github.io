# Kas Cranky | Site Oficial

Site estático de biografia e portfólio de **Francisco Armando Chico (Kas Cranky)**, com foco em apresentação pessoal, carreira e redes sociais.

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)
- Font Awesome (ícones)

## Funcionalidades implementadas

- Hero com efeito de digitação no título.
- Carrossel de 3 fotos com:
  - autoplay;
  - navegação por setas e bolinhas;
  - transição suave entre slides.
- Lightbox ao clicar na foto (ampliação em até 80% da tela).
- Skeleton loader nas imagens enquanto carregam.
- Fundo com imagem (`backgound.jpg`) + overlay escuro + blur.
- Botão flutuante `Topo` com rolagem suave de 1 segundo.
- Navbar flutuante de atalhos (aparece após scroll).
- Links de redes sociais abrindo em nova aba.
- Favicon configurado com `me.png`.
- Lazy loading aplicado para imagens fora da primeira dobra.

## Estrutura do projeto

- `index.html`: estrutura e conteúdo da página.
- `style.css`: layout, temas visuais, animações e responsividade.
- `script.js`: interações (typing, slider, lightbox, scroll, skeleton).
- `me.png`: favicon e imagem de compartilhamento.
- `IMG_3616.jpg`, `slide1.jpg`, `slide2.jpg`: imagens do carrossel.
- `backgound.jpg`: fundo global do site.

## Como executar localmente

1. Abra a pasta do projeto.
2. Execute um servidor local simples (opcional, recomendado).
3. Abra `index.html` no navegador.

Exemplo com VS Code: usar a extensão **Live Server**.
